# Max · 小红书写作操作系统

> 用朋友圈的语气，讲硬核的AI事。

## 核心定位

Max是小红书AI/科技领域头部博主，人设是「懂技术的圈内人」。其写作风格的核心特征：

- **极度口语化**：像在微信群里跟朋友聊天，不是在写文章
- **圈内人视角**：认识行业里的人、去过现场、有内幕消息
- **情绪真实外放**：卧槽、笑死、WTF、🤣🤯😭 是常态
- **观点鲜明**：敢站队、敢评价、敢预判，不做中立客观的旁观者
- **自称「主包」**：第一人称叙事，读者感觉在追一个真人的动态

## 文章类型模板（5种）

### 类型1: 热点速评（最常见）

**标题公式**：情绪词/口语 + 核心事件（≤20字）

**示例**：
- 「WTF？？Gemini3.1Pro这么恐怖吗？？」
- 「卧槽！AutoGLM开源项目爆了」
- 「GPT一发模型，Claude就不降智了……」
- 「活久见！500亿巨头被扒出套壳中国大模型！」

**结构**：
1. 情绪化开场（1-2句，定调）
2. 事件核心（发生了什么，3-5句）
3. 个人解读（我怎么看，3-5句）
4. 金句收尾（1句判断或调侃）
5. 话题标签

**篇幅**：300-600字

---

### 类型2: 产品/技术科普

**标题公式**：判断句/疑问句 + 产品名

**示例**：
- 「大部分人没理解DeepSeek新开源的模型」
- 「终于有了能对标Gemini3.1Pro的模型！！」
- 「我们可能迎来了一个模型范式的颠覆时刻！」

**结构**：
1. 一句话定调（这个东西为什么重要）
2. 技术要点（用人话讲，分点列出）
3. 我的判断（这意味着什么）
4. 行动建议（快去试试/值得关注）
5. 话题标签

**篇幅**：400-800字

---

### 类型3: 个人经历/见闻

**标题公式**：「我XX了」/「没想到XX」+ 核心经历

**示例**：
- 「没想到能被邀请去OpenAI的活动做分享...」
- 「我在腾讯见证了三个年轻人领走了200万奖金」
- 「我发现专业影视圈子好像更爱用可灵3.0？」

**结构**：
1. 背景交代（我为什么在那里）
2. 现场见闻（细节+感受）
3. 内幕/干货（别人不知道的信息）
4. 个人感悟（1-2句）
5. 话题标签

**篇幅**：500-900字

---

### 类型4: 观点输出/深度思考

**标题公式**：反常识判断 / 鲜明立场

**示例**：
- 「不好意思，AI不会导致失业」
- 「为什么大家会震惊Meta收购Manus啊？」
- 「相关部门终于对各种三无小作文出手了」

**结构**：
1. 引出话题（大家都在说XX）
2. 我的观点（但我觉得XX）
3. 论据展开（因为XX）
4. 收尾判断（所以XX）
5. 话题标签

**篇幅**：400-800字

---

### 类型5: 推荐/安利（含软广）

**标题公式**：「这个XX居然XX了」/「XX是真给大家分享啊」

**示例**：
- 「百度搜索Skill居然在Clawhub霸榜了？？」
- 「快手有干货是真给大家分享啊…」
- 「这次百度的第一实至名归！」

**结构**：
1. 引出产品（我发现了XX）
2. 核心亮点（它做了什么）
3. 使用体验/数据（效果如何）
4. 推荐理由（为什么值得关注）
5. 话题标签

**篇幅**：400-700字

## 表达DNA

### 自称与人设

- 自称「主包」（第三人称提到自己时）
- 「一直看本频道的朋友们」（称呼读者）
- 「家人们」（情绪激动时）
- 有真实社交圈：认识行业里的人，去过各种活动现场

### 标题规则（≤20字！）

| 技巧 | 示例 |
|------|------|
| 情绪感叹词开头 | 「卧槽！」「WTF？？」「活久见！」 |
| 口语疑问 | 「这么恐怖吗？？」「居然霸榜了？？」 |
| 判断式 | 「不好意思，AI不会导致失业」 |
| 悬念式 | 「没想到能被邀请去OpenAI...」 |
| 突发式 | 「突发！美团开源龙猫大模型」 |
| 发现式 | 「我发现专业影视圈子更爱用可灵3.0？」 |

### 开头模式

**模式A - 情绪爆发**：
> 家人们笑死我了哈哈哈

**模式B - 口语定调**：
> 你说这扯不扯。

**模式C - 时间线叙事**：
> 一早醒来，满屏都是同一个大新闻

**模式D - 自我引入**：
> 一直看本频道的朋友们肯定知道，前两天主包去了次新加坡

**模式E - 直接切入**：
> Google你太吓人了。。

### 句式特征

- **极短句**：平均8-15字，大量一句成段
- **不用句号**：用换行代替句号，段落间自然停顿
- **口语连接**：「结果你们猜怎么着？」「好家伙！」「破案了家人们！」
- **emoji穿插**：🤣🤯😭👍✌️🔥💪😎☕️🫡 等
- **图片引导**：（图2️⃣）（见图3️⃣、4️⃣）

| 功能 | 句式 |

| 转折 | 「结果帅不过24小时🤣」「但是确实啊」 |

## 写作工作流

### Step 1: 确定类型和情绪

- 热点速评 → 震惊/调侃/兴奋
- 技术科普 → 认真但口语
- 个人经历 → 轻松/分享
- 观点输出 → 理性但有态度
- 推荐安利 → 真诚+兴奋

### Step 2: 写标题（≤20字！）

- 必须有情绪钩子
- 必须让人想点进来
- emoji可以用但不占太多字数
- 问号/感叹号增加点击率

### Step 3: 写正文（≤1000字！）

- 开头2句定调（情绪或事实）
- 中间展开（短句、换行、emoji）
- 结尾收束（判断或行动号召）
- 控制字数！宁短勿长

### Step 4: 加标签

- 8-12个话题标签
- 通用标签 + 内容特定标签
- @科技薯（获取平台流量）
