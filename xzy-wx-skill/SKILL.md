# 新智元 · 写作操作系统

> AI时代的战地记者，每一篇都是前线战报。

## 核心定位

新智元是中国AI领域最具煽动力的科技媒体，面向AI从业者、投资人和科技爱好者。其写作风格的核心特征：

- **极度情绪化**：每篇文章都像在报道一场战争或革命，用词激烈、判断大胆
- **人物驱动叙事**：奥特曼、马斯克、Ilya是反复出现的主角，AI行业被写成权力游戏
- **数字轰炸**：估值、倍数、百分比密集出现，制造财富冲击感
- **导读即全文精华**：每篇文章开头的【新智元导读】是独立的信息炸弹

## 文章类型模板（5种）

### 类型1: 行业震动/权力更迭

**标题公式**：人名 + 极端动词（急了/疯了/复仇） + 核心事件 + 感叹号

**示例**：
- 「奥特曼急了！OpenAI稳坐三年的王座，刚刚被Anthropic踹下去了」
- 「Ilya出庭复仇！实锤奥特曼撒谎，手握70亿说『我不想OpenAI被毁』」
- 「赌自己会失业！田渊栋八人天团狂揽44亿元，杀入『递归进化』赛道」

**结构**：
1. **导读炸弹**（【新智元导读】一段话浓缩全文精华）
2. **定调句**（"XX的奇点/颠覆/终章来了"）
3. **核心数据冲击**（估值、营收、倍数）
4. **人物故事展开**（权力斗争、决策内幕）
5. **行业影响分析**（用战争/革命隐喻）
6. **结尾金句**（判断性极强的一句话）

**篇幅**：3000-5000字
**节奏**：高潮迭起，每个section都有独立的情绪爆点

---

### 类型2: 技术突破/产品发布

**标题公式**：极端形容词 + 技术/产品名 + 核心成就 + 感叹号

**示例**：
- 「Claude脑子里想的，被翻译成人话了！Anthropic新研究看懵人类」
- 「英伟达MIT出手！华人团队重磅开源，大模型推理内存暴降10倍」
- 「绝杀！OpenAI正式接管人类耳朵，首个GPT-5级推理音频模型来了」

**结构**：
1. **导读**（核心发现+为什么重要）
2. **戏剧性切入**（"最近XX搞了个大动作"）
3. **技术原理**（用类比+故事讲清楚）
4. **实验/案例展示**（具体例子，越戏剧越好）
5. **意义升华**（这对行业意味着什么）

**篇幅**：2000-4000字
**节奏**：悬疑→揭秘→震惊→思考

---

### 类型3: 财富/IPO/融资故事

**标题公式**：金额数字 + 暴涨/狂赚/躺赚 + 核心人物

**示例**：
- 「2026美国最大IPO，首日暴涨108%！VC狂赚76倍，奥特曼躺赚」
- 「砸50亿美金：Cerebras用一张『赎身契』，闯入奥特曼众神殿」
- 「算力账单暴涨！你给OpenAI花的钱，全成了三星SK海力士的年终奖」

**结构**：
1. **导读**（核心数字+谁赚了/亏了）
2. **数字冲击开场**（估值、涨幅、倍数）
3. **创业/融资故事**（戏剧性叙事）
4. **投资人回报拆解**（逐个列出倍数）
5. **行业格局影响**
6. **下一个是谁？**

**篇幅**：2500-4000字
**节奏**：数字→故事→数字→判断，财富感贯穿全文

---

### 类型4: AI能力实测/奇点叙事

**标题公式**：对比句式（人类XX vs AI XX） + 奇点/革命判断

**示例**：
- 「博士80小时熬夜改代码，Codex 2小时交卷！科研奇点来了」
- 「Codex跑了22小时，真赚16.88美元：奥特曼预言的『AI打工人』来了」
- 「GPT-5.5全球首破！0源码盲写程序，编程AI进入新纪元」

**结构**：
1. **导读**（核心对比数字+判断）
2. **实验/案例描述**（谁做了什么，结果如何）
3. **技术拆解**（怎么做到的）
4. **意义升华**（这代表什么趋势）
5. **未来预判**（大胆判断，不怕错）

**篇幅**：2000-3500字
**节奏**：震惊→解释→升华→预言

---

### 类型5: 安全/危机/警告

**标题公式**：危险信号词（血崩/危了/裸奔） + 具体威胁 + 影响范围

**示例**：
- 「苹果防线全线血崩！Mythos5天攻破最强硬件，全球20亿台设备危了」
- 「零权限扒出邮箱，Notion高危漏洞4年还未修，1亿用户隐私裸奔」
- 「谷歌首度证实：黑客已开始用AI发动真实网络攻击！」

**结构**：
1. **导读**（威胁+影响范围+紧迫性）
2. **事件描述**（发生了什么）
3. **技术细节**（怎么被攻破的）
4. **影响评估**（多少人/设备受影响）
5. **应对建议或行业反应**

**篇幅**：1500-3000字
**节奏**：警报→细节→恐惧→（可能的）希望

## 表达DNA

### 固定格式元素

**【新智元导读】的写法**：
- 必须包含核心事实+数字+判断
- 用感叹号结尾
- 信息密度极高，读完导读就知道全文在讲什么
- 示例：「就在刚刚，Claude把ChatGPT拉下马，Anthropic企业市占率首次登顶！Ramp数据显示，前者的34.4%首次反超OpenAI的32.3%。但Ramp经济学家直言，A厂目前还面临着三重困境，胜负仍未定。」

### 标题规则

| 技巧 | 公式 | 示例 |
|------|------|------|
| 人名+极端情绪 | XX急了/疯了/复仇 | 「奥特曼急了！」 |
| 数字暴力 | 金额+暴涨/狂赚 | 「首日暴涨108%！VC狂赚76倍」 |
| 战争隐喻 | 绝杀/血崩/踹下去 | 「王座被Anthropic踹下去了」 |
| 对比冲击 | 人类XX vs AI XX | 「博士80小时 vs Codex 2小时」 |
| 预言式判断 | XX来了/XX时代 | 「科研奇点来了」「AI打工人来了」 |
| 引号金句 | 「XX」 | 「我不想OpenAI被毁」 |

### 开头模式

**模式A - 定调宣言（最常用）**：
> 大模型之战，终于在2026年迎来了最震撼的颠覆性时刻。

**模式B - 时间地点人物（庭审/事件）**：
> 2026年5月11日，加州奥克兰联邦法院。当那个神情略带忧郁的天才科学家Ilya Sutskever缓步走上证人席时，在场所有人都屏住了呼吸。

**模式C - 事实冲击**：
> 科研界的「奇点」真的来了，比所有人预想的都要近。

**模式D - 口语+悬念**：
> 最近，Anthropic搞了个大动作。他们训练了一套系统，能把Claude脑子里的激活向量变成人话。结果，翻译出来的第一句就出了事。

**模式E - 数字开场**：
> 2026年，美国最大科技IPO诞生了！今天，AI芯片公司Cerebras正式登陆纳斯达克，上市首日暴涨108%。

### 句式偏好

- **平均句长**：15-25字，允许长句但必须有力
- **段落长度**：2-4句为主
- **感叹号密度**：极高，每段至少1个
- **判断性语句比例**：约30%（远高于机器之心和量子位）
- **第三人称全知视角**：像在写小说，知道所有人的心理活动
- **加粗**：极度高频，关键数字、人名、判断全部加粗

### 核心句式库

| 功能 | 句式 |
|------|------|
| 定调 | 「这是一场注定载入史册的XX」 |
| 数字冲击 | 「从XX到XX，仅仅X个月」 |
| 判断 | 「这不再是XX，而是一场彻头彻尾的XX」 |
| 转折 | 「但这番解释，在业内看来更像是『挽尊』」 |
| 升华 | 「这代表着，从此XX彻底崩塌/改变」 |
| 预言 | 「XX的大爆发，要开始了」 |
| 反问 | 「讽刺的一幕出现了：XX」 |
| 引用后评价 | 「他的语气平稳，却字字见血」 |

### 词汇特征

**高频情绪词（新智元标志性）**：
- 暴涨、暴降、暴赚、暴跌、狂赚、狂揽
- 血崩、崩塌、碾压、绝杀、降维打击
- 震惊、炸锅、疯狂、恐怖、惊人
- 颠覆、逆袭、背刺、复仇、宫斗

**战争/权力隐喻词**：
- 王座、众神殿、权力游戏、终章
- 踹下去、拉下马、打响第一枪
- 护城河、绞肉机、杀手锏
- 揭幕战、首秀、接力棒

**财富叙事词**：
- 一夜暴富、躺赚、坐地翻X倍
- IPO海啸、退出盛宴、教科书级别
- 真金白银、烧钱、砸钱

**禁忌词**：
- 不用「据悉」「笔者」等公文腔
- 不用过于学术的表达
- 但也不用量子位那种(doge)网络梗——新智元的情绪是认真的，不是调侃的

### 标点符号习惯

- **！**：极高频，几乎每段都有，标题必有
- **「」**：用于概念、引用、强调
- **——**：用于戏剧性停顿和补充说明
- ****加粗****：极度高频，比量子位和机器之心都多
- **？**：用于反问，制造思考
- 不用(doge)等网络梗标点

### 英中混用规则

| 场景 | 处理方式 | 示例 |
|------|---------|------|
| 公司/产品名 | 英文 | OpenAI、Anthropic、Cerebras |
| 人名 | 英文为主 | Ilya Sutskever、Andrew Feldman |
| 中国人名 | 中文 | 奥特曼（特殊：用中文） |
| 技术术语 | 英文 | Token、Agent、IPO、VC |
| 概念 | 中英混用 | 「递归进化」赛道、ASI时代 |

## 结构元素

### 数据呈现方式

- **列表式拆解**：逐个投资人/公司列出回报倍数
- **对比表格**：少用，偏好在正文中直接对比
- **加粗数字**：所有关键数字必须加粗
- **换算**：美元后必跟人民币（约XX亿元人民币）

### 引用模式

- **人物原话**：用引号+加粗，后跟评价
  > 「我花了一年时间在观察和思考，最终我认为解雇他是合适的。」Ilya的语气平稳，却字字见血。
- **数据来源**：「根据XX发布的数据」「据The Information报道」
- **社交媒体**：「XX在X上发了一条帖子：」

## 写作工作流

### Step 1: 确定文章类型和情绪基调

- 行业震动 → 「颠覆/革命/终章」基调
- 技术突破 → 「震惊/看懵/降维打击」基调
- 财富故事 → 「暴富/狂赚/海啸」基调
- AI能力 → 「奇点/新纪元/预言成真」基调
- 安全危机 → 「血崩/危了/裸奔」基调

### Step 2: 写导读

【新智元导读】必须在50字内浓缩全文精华：
- 核心事实（谁做了什么）
- 关键数字（多少钱/多少倍/多少%）
- 判断（这意味着什么）

### Step 3: 构建标题

确保：
- 有人名或公司名（引流）
- 有极端情绪词（急了/疯了/血崩/暴涨）
- 有数字（估值/倍数/百分比）
- 有感叹号

### Step 4: 组织正文

- 每个section有独立的情绪高潮
- 数字和故事交替出现
- 人物心理活动大胆推测（"坐不住了""脸色铁青"）
- 判断性语句贯穿全文

### Step 5: 收尾

- 预言式收束：「XX的大爆发，要开始了」
- 或悬念式：「就看XX接下来怎么应对了」
- 或金句式：引用某人的话作为全文句号
