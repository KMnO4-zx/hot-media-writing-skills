# 🔥 AI科技媒体写作Skills

基于中国头部AI科技媒体的真实文章蒸馏而成的写作风格技能包。每个Skill从近期文章中提炼出完整的写作操作系统，包含文章模板、表达DNA、标题公式和结构框架。

## 四个Skill一览

| Skill | 平台 | 风格定位 | 适合场景 |
|-------|------|---------|---------|
| `jqzx-wx-skill` | 微信公众号 | 专业深度，技术翻译官 | 论文解读、技术拆解 |
| `lzw-wx-skill` | 微信公众号 | 口语玩梗，朋友聊天 | 产品体验、热点快讯 |
| `xzy-wx-skill` | 微信公众号 | 情绪判断，战地记者 | 行业震动、融资人物 |
| `max-xhs-skill` | 小红书 | 圈内人视角，群聊语气 | 短平快速评、个人见闻 |

## 风格光谱

```
专业克制 ←————————————————————→ 情绪外放
机器之心    量子位    新智元    Max(小红书)
```

## 使用方式

将对应skill文件夹放入 `.claude/skills/` 或 `.agents/skills/` 目录，通过触发词激活：

- 「用机器之心风格写一篇关于XX的文章」
- 「用量子位风格写」
- 「用新智元风格写」
- 「用Max风格写一篇小红书」

## 各Skill核心差异

**机器之心** — 让技术有温度，让前沿可触达
- 署名：编辑｜XX
- 句长：15-30字，段落3-5句
- 特征：「」强调术语、类比解释技术、数据表格对比
- 禁忌：网络梗、情绪化判断

**量子位** — 追踪人工智能新趋势
- 署名：XX 发自 凹非寺 量子位 | 公众号 QbitAI
- 句长：10-20字，大量一句成段
- 特征：第一人称、(doge)梗、One More Thing彩蛋
- 禁忌：公文腔、长段落

**新智元** — AI时代的战地记者
- 署名：新智元报道 + 【新智元导读】
- 句长：15-25字，判断性语句占30%
- 特征：战争隐喻、数字轰炸、人物权力叙事
- 禁忌：中立客观、网络梗

**Max小红书** — 用朋友圈的语气讲硬核AI事
- 格式：标题≤20字，正文≤1000字，文末话题标签
- 句长：8-15字，不用句号用换行
- 特征：emoji高频、自称主包、圈内人内幕
- 禁忌：书面语、长句、无态度
